# Usage

### Clone

`git clone git@github.com:restaurant-management/resman-mobile-customer.git`

`cd resman-mobile-customer`

### Get package

`flutter pub get`

### Add your **google-sevices.json** to **android/app/**

### Apply launcher icon

`flutter pub run flutter_launcher_icons:main`

### Run

`flutter run`
